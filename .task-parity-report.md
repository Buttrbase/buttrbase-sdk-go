### Usage Reporting

| Method | Verb | Endpoint | Notes |
|---|---|---|---|
| `ReportUsage(ctx, event)` | POST | `/api/usage/report` | Body: `UsageEvent{Metric, Quantity, OrgUUID?, AppUUID?, Timestamp?}` |

**Divergence (documented):** Rust SDK uses HTTP Basic auth (client_id:client_secret) via `app_request()` for this endpoint. The Go client uses an OAuth2 bearer token model; there is no per-request Basic auth path. `auth=true` (bearer) is used. This is consistent with how all other app-level calls work in the Go SDK.

If the server treats both paths equivalently, this is a no-op difference; if not, callers should migrate to the `Typed` variant.
